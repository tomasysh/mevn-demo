# mevn-demo

# M: MySQL
# E: Express
# V: Vue
# N: NodeJS

本專案示範基本的前後端分離，後端專案提供下列 api:
/postLogin
/postSignup

使用方式：

後端專案：
1. 進入專案資料夾後，再進入後端資料夾 backend-project。
2. 執行 npm install 安裝套件。
3. 執行 node app.js 起 Web Server(poer 4000)。

前端專案：
1. 進入專案資料夾後，再進入後端資料夾 vue-project。
2. 執行 npm install 安裝套件。
3. 執行 npm run dev 起前端專案 (port 3000)。
